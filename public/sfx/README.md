# 효과음 (선택 사항)

배포 환경에서 외부 URL이 막혀 소리가 안 나면, 아래 파일을 이 폴더에 넣어주세요.

- `click.ogg` — 클릭음
- `transition.ogg` — 문구 전환음
- `success.ogg` — 임명장 완료음
- `alert.ogg` — 불효자 경고음

다운로드 예시 (브라우저에서 링크 열고 우클릭 → 다른 이름으로 저장):

- click: https://actions.google.com/sounds/v1/button/button_interactive_single.ogg
- transition: https://actions.google.com/sounds/v1/foley/wind_swoosh.ogg
- success: https://actions.google.com/sounds/v1/cartoon/clime_up_the_ladder.ogg
- alert: https://actions.google.com/sounds/v1/cartoon/boing.ogg

파일명을 위와 같이 맞춰 저장한 뒤 `public/sfx/` 에 넣으면 됩니다.
